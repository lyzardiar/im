#This is the help for client

Connect to server : ./client hostip your_nickname

When connected:
	-user or -u		: query online users from server
	
	-all or -a		: set chat mode to speak to everyone
	
	-private user_nickname or -p user_nickname	
					: set chat mode to speak to user nickname
					
	-list num or -l	num
					: set message list size to num, if num == 0 or invaliable, set it
						to default value 8
	
	/user_nickname msg		: send msg to user_nickname only
				
	-help or -h		: get this help 
	
	-quit or -q		: quit from server
